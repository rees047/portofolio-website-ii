POKEMON

Objective: The objective of this project is to display the capability of html, css and boostrap and jquery in simple projects while using an external API to get data.

Featured Techonologies: HTML, CSS, JQuery

User Story:
As a user,
I want to click on the pokeball
so that I can see data pokemon data

Acceptance Criteria:
Given the user has chosen desired pokeball to click
When the user clicks on the pokeball
Then the pokeball opens and shows data about chosen pokemon

MEET MY CITY

Objective: The objective of this project is to build a serverless, progressive web application (PWA) with React using a test-driven development (TDD) technique. This application uses the google Calendar API to fetch upcoming events.

Featured Techonologies: React

User Story:
As a user,
I want to choose a city
so that I can see the upcoming events within that city

Acceptance Criteria:
Given the user types in a city
When the user chooses a city from the suggestion box
Then events related to the chosen city are shown

Github Repository: Meet my City Repository

Live Demo: Meet My City Demo
